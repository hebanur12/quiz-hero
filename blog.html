1. Different Map ,Filter , ForEach , Find

  1. Map -- Transform each item in an array and output to a new array without altering the original array.
  2. Filter -- remove items of an array based on some criteria without altering the original array.
  3. forEach -- perform some function on each item of an array.
  4. find -- locates a specific entry by a predicate and returns it.

2. Different Local Storage and Session Storage

Local storage maintains data across all instances of a site, whether they're in different tabs or windows. It is also permanent, so the data won't disappear when you close your browser. When you open a website that previously used local storage, it will always have access to that stored data.


Session Storage is cleared when the page session ends. A page session lasts as long as the tab or the browser is open. Therefore, when you open a new tab or a new window, this creates a new session and new storage.

3. Event Loop

The event loop is the secret behind JavaScript's asynchronous programming. JS executes all operations on a single thread, but using a few smart data structures, it gives us the illusion of multi-threading.
